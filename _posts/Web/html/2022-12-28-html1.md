## 💚 HTML 페이지의 기본

```
<!DOCTYPE html>
<html>
<head>
  문서 제목, 자바스크립트 코드, CSS 스타일 정의, 메타 데이터 정의
<head>
<body>
  문서의 본문 텍스트, 이미지, 테이블, 자바스크립트 코드, 동영상 등
</body>
</html>
```
